# Apollo

## Discord Bot to display Arma 3 server info
The bot provides displays info about the:
- Server title
- Steam link to connect
- Player count
- Game mode
- Map
- Password protected


## Requirements
- Node >= 8.11.3
- NPM >= 5.6.0


## Setup
- Get the code
- `npm install`
- `cp .env.example .env`
- Configure the `.env` file
- `node bot.js`
- Good to go!


## Available commands
- `!update` to quickly get info about the server status.
